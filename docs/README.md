# Pixi Paper

Real-time comparison of HTML to Canvas/Image screenshot libraries.

## Live Demos

- [Home Demo](index.html) - Basic HTML to canvas rendering
- [Library Comparison](compare.html) - Side-by-side comparison of html2canvas, html-to-image, modern-screenshot, and Satori
- [Stress Test](stress.html) - Performance testing with complex layouts

## Libraries Compared

| Library | Version | Approach |
|---------|---------|----------|
| html2canvas | 1.4.1 | DOM to Canvas |
| html-to-image | 1.11.11 | SVG foreignObject |
| modern-screenshot | 4.5.0 | Multiple backends |
| Satori | 0.18.3 | JSX to SVG |
